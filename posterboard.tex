    \begin{column}{0.5\linewidth}

      \textbf{YES/NO Securities}
      \begin{itemize}
        \item Price $\in [0,1]$ represents implied probability
        \item YES security pays \$1 if event occurs
        \item NO security pays \$1 if event does not occur
      \end{itemize}

      \textbf{Probability Space}
      \begin{itemize}
        \item $(\Omega, \mathcal{F}, P)$: sample space, events, probability measure
        \item $\Omega$: all possible states of the world
        \item Filtration $\mathbb{F} = (\mathcal{F}_t)$: information flow over time
      \end{itemize}

      \textbf{Martingale}
      \begin{itemize}
        \item Model for a fair game
        \item $E[X_{t+1} | \mathcal{F}_t] = X_t$
        \item Current price equals expected future value
      \end{itemize}

      \textbf{Brownian Motion}
      \begin{itemize}
        \item Continuous-time stochastic process
        \item $W(t) \sim N(0, t)$ with continuous paths
        \item $\sqrt{\Delta t}$ scaling for variance consistency
      \end{itemize}

    \end{column}

  \begin{block}{Model II: Earnings Model}


    \textbf{Signal Components:}

    \begin{itemize}
      \item Polymarket price (probability) of beating earings  
      \item StarMine SmartEstimates EPS Score
      \item 200 days of analyst recommendation
    \end{itemize}


    If all components agree, place a trade on that direction.


    \textbf{Position Sizing:}

    Assign a weight to each component based on predictive power, $w_1, w_2, w_3$ such that $w_1 + w_2 + w_3 = 1$. \\
    Polymarket price is normalized to [-1,1], by taking price (p - 0.5) * 2, then scaled by volume. 
    Volume Scale = max( $\frac{log_{10}(volume)}{5}$, 1)


    Smart Estimate 


    analyst recommendation are a range of 1,5 so to nromaile it subtract 3 form the mean socre and divide by 3 
    $\frac{A_{rec} - 3}{2}$

    $x_1 * w_1 + x_2 * w_2 + x_3 * w_3 \in [-1,1] $







  \end{block}
